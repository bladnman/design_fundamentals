# Design Fundamentals

An interactive slideshow that teaches core design principles through hands-on demos. Each slide lets you manipulate design properties and see the impact in real-time.

## Getting Started

```bash
npm install
npm run dev
```

## The 15 Slides

| Slide | Concept | What You Learn |
|-------|---------|----------------|
| **Intro** | Welcome | Overview of the fundamentals |
| **Tone** | Lightness, temperature, saturation | How mood comes from color choices |
| **Contrast** | Foreground vs background | Directing attention through visual weight |
| **Hierarchy** | Information priority | Making the most important thing stand out |
| **Spacing** | Padding, gaps, line height | How breathing room affects readability |
| **Alignment** | Grid structure, chaos mode | Why invisible lines create trust |
| **Typography** | Scale types, ratios | How size relationships create rhythm |
| **Fonts** | Personality through typeface | When to use serif, sans, display fonts |
| **Color** | Harmony types | Monochromatic, complementary, analogous palettes |
| **Elevation** | Shadows and depth | What floats, what's grounded |
| **Emphasis** | Drawing attention | Size, color, position as emphasis tools |
| **Repetition** | Consistency | Why similar things should look similar |
| **Rhythm** | Sequence patterns | Guiding users through multi-step flows |
| **Affordance** | Interactivity cues | Making clickable things look clickable |
| **Restraint** | Less is more | When adding hurts more than it helps |

## Navigation

- **Arrow keys** or **A/D** to move between slides
- Click the slide dots at the bottom

## Key Takeaways

Each slide includes:
- **Why it matters** - A brief explanation of the concept's real-world impact
- **Interactive demo** - Manipulate properties to see the principle in action
- **Diagnostic quote** - A memorable heuristic for identifying problems

## Tech Stack

- React + Vite
- Tailwind CSS
- Custom font stack (Inter, Playfair Display, Space Grotesk, etc.)
